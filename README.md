# nodejs-authentication-app
Simple nodejs authentication app using express, passport and passport-local

To be able to use this app;
*** node should be installed must be installed on your computer.
*** mongodb should also be installed on your computer.

For the app to work, the following npm packages, must be installed in the root directory of the app:
1. body-parser
2. ejs
3. express
4. express-session
5. mongoose
6. passport
7. passport-local
8. passport-local-mongoose

npm packages can be installed using the following methods

1. navigate to the root directory of the app in cmd
2. type 'npm init'
3. follow the prompt correctly
4. then type 'npm install <npm package name> --save'
	e.g to install ejs type 'npm install ejs --save';
	repeat step 4 for all the required listed packages above;

To run this app 
***From the node console at the root directory of the app type 
	'node app.js';
